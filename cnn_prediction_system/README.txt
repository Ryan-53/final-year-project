Python version==3.9.13
pip version==24.0

To run program:

1. Install requirements stored in requirements.txt:

    pip install -r requirements.txt

    [This may fail to install the correct version of tesorflow due to a longpath error
    - If so you will need to enable longpaths in your system settings]

2. Run batch file prediction_app.bat to start application

  Alternatively you can run python app.py in the command line
